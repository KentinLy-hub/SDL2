##Project_SDL_C++
    ~ Build project for Linux

1 . Build the project
    In the root folder, run `make` commande.
        $ mkdir build && cd build
        $ cmake ..    

2 . Execute the project
    In 'build' folder :
        run `make` commande
        ~~ ./Project_SDL_Part1_base/SDL_part1 {n_sheep} {n_wolf} {second_exec}` ~~       
            $ ./Project_SDL_Part1_base/SDL_part1 4 3 2
        
3 . Clean the project
    In 'build' folder :
        $ `make clean`
